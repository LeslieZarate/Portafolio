MD - LINKS

NODE JS | JAVASCRIPT

Estamos leyendo sobre un tema interesante en un blog ;foro o el readme de algún repositorio de GitHub y encontramos un enlace ,nos dirigimos a este y nos muestra una pantalla de error ,el típico 404. Pero ,¿ Qué tienen en común este tipo de fuentes de información ?.Muchas de ellas utilizan archivos del formato Markdown.

Md-Links ,es una libreria que permite leer y analizar archivos de formato Markdown, para verificar los links que contenga este y reportar estadísticas sobre estos.

"No permitas que se pierda el valor de la información que quieres compartir, solo por links rotos"
